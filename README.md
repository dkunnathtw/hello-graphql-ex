# Build

```
./gradlew clean build
```

# Run

```
./gradlew bootRun 
```

or, following build command above

```
java -jar build/libs/hello-graphql-1.0.0-SNAPSHOT.jar
```

# Testing

Access GraphiQL UI at [http://localhost:8080/tools/graphiql](http://localhost:8080/api/graphiql)
